# 🐾 动物王国 · 趣味知识库

## 项目介绍
这是一个关于动物趣味知识的互动网页应用，包含50+条动物冷知识、20种动物图鉴和35道知识挑战题。用户可以在网页上随机浏览动物趣闻、按分类筛选内容，并通过答题挑战测试自己的动物知识水平。

## 项目用途
本项目用于学习和实践 GitHub 基础操作，包括创建仓库、编写 README 文件、上传代码文件等。同时也是一个有趣的前端小项目，适合动物爱好者和编程初学者参考学习。

## 使用方法
1. 打开仓库中的 `index.html` 文件，即可在浏览器中运行本项目
2. 点击页面上的标签按钮（趣味趣闻/动物图鉴/知识挑战）切换不同功能模块
